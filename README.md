## What do you need?

- Gradle
- Java 8

## To compile

Run:

```
$ gradle clean build
:clean
:compileJava
...
:build

BUILD SUCCESSFUL

Total time: 8.621 secs
```

### To run

Run the jar:

```
$ java -jar build/libs/AppMoney.jar
```
